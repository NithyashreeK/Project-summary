Project #1: Game of life

Game of life, also known as Conway's game, is an orthogonal grid of square cells which is two dimensional and the cells have either of the two states, alive or dead. Transition between these two states occurs when:
-Any live cell with fewer than two live neighbours dies, as if caused by underpopulation.
-Any live cell with two or three live neighbours lives on to the next generation.
-Any live cell with more than three live neighbours dies, as if by overpopulation.
-Any dead cell with exactly three live neighbours becomes a live cell, as if by reproduction.
Wrap-around for the edges was used. (Ex. (-1 + 10) % 10 = 9)
Implementation of this project was done by using p5.js, a Javascript library.
